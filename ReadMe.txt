simple  Java script that reads in a list of Civ5 civilizations, their units and unique abilities
and sorts them randomly into n groups depending on how many players are in the game.

cmdline usage:
java -jar {number of civs per player <=10} {name of p1} {name of p2}

e.g.
java - jar 3 jack malcolm matt

Output:
Creates 1 txt file output for each player, containing his civs to choose from.
